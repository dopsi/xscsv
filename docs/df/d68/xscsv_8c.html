◆ strtok_protect()

static char* strtok_protect ( char *  s,
const char *  delim,
char  protection 
)
static

strtok variation with substring protection

This function is called in the same way strtok is. On the first call, it take the string to be split, the delimiters and the protection character as arguments. On the following call, s must be NULL and the tokens will be returned until there is no token left, at which point it returns NULL;

char *pch = strtok_protect("a,\"b,c\",d\0", ',', '"'); // returns 'a'
while (pch != NULL) {
pch = strtok_protect(NULL, sep, protector); // return '"b,c"'
// 'd'
// and NULL
}
Parameters
sstring to split
delimlist of delimiters
protectionprotection character
Returns
the current token
