xscsv

Usage

All functions and type definitions are located in xscsv.h

#include <xscsv.h>

A document (which is nothing but a pointer to a xscsv_document_t structure) is opened with xscsv_open. The `'\0'` parameter

const char filename[] = "example.csv";
xscsv_document_t *doc = xscsv_open(filename, '\0');

If you already have a FILE* pointer, this file can be read with xscsv_read.

xscsv_document_t *doc = xscsv_read(file_pointer, '\0');

To create an empty document, use xscsv_new.

xscsv_document_t *doc = xscsv_new('\0');

From this point, all three pointers are functionnaly equal.

To access document properties, use xscsv_lines, xscsv_columns or .

size_t lines_in_document = xscsv_lines(doc);
size_t max_columns_in_document = xscsv_columns(doc);
size_t columns_in_first_line = xscsv_columns_in_line(doc, 0);

To close the document properly, xscsv_close is used

xscsv_close(doc);
doc = NULL;

For more example programs, see the examples/ directory.
